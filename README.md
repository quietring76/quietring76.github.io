## 📝 第一堂课：大数据驱动的科学

### 1. 核心概念辨析：模型 (Model) vs. 算法 (Algorithm)

在第一节课中，我们明确了生物信息学的四个步骤：**问题 -> 信息 -> 分析 -> 建模**。其中，“建模”阶段最容易混淆模型与算法的区别：

* **模型 (Model)**：
    * **定义**：是对现实生物学问题的数学抽象或逻辑表达。
    * **本质**：它是一种“结构”或“框架”，代表了我们对规律的理解。
    * **例子**：**回归模型 (Regression)**（如用于分类的 Logistic 回归）或 **神经网络模型 (Neural Network)**（深度学习的核心）。
* **算法 (Algorithm)**：
    * **定义**：是解决特定计算问题的具体步骤或指令集。
    * **本质**：它是一种“工具”或“方法”，用于训练模型或处理数据。
    * **例子**：**排序算法 (Number Sorting)** 或生物信息学中最经典的 **动态规划算法 (Dynamic Programming)**。

**总结**：模型是我们要构建的数学“房子”，而算法是建造房子的“施工手册” 。
